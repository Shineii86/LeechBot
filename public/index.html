Getting Started

LeechBot is an advanced Telegram file transloader that allows you to download and upload files via Telegram. This interface helps you configure and deploy the bot on Google Colab.

Required Credentials:

  • API ID & API Hash: Obtain from my.telegram.org
  • Bot Token: Create via @BotFather on Telegram
  • User ID: Your personal Telegram user ID
  • Dump ID: Channel or group ID for logging (format: -100XXXXXXXXXX)

⚠️ Important

  • Never share your API credentials or bot token publicly
  • Google Colab runtime will disconnect after inactivity
  • Keep the keep-alive mechanism active during deployment
  • Ensure your bot has appropriate permissions in the dump channel
